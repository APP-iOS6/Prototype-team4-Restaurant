# 앱 이름
작성자의 누적 방문 횟수를 기반으로 맛집 리스트를 확인하고,
좋아요를 누른 가게 근처로 가게되면 알림을 받게 하는 앱

# Persona
- 해당 지역을 처음 / 재방문 했는데, 맛있는 음식점을 가고 싶은 사람
- 맛집 방문하는것을 좋아하지만, 광고로 도배되어 신뢰가 떨어지는 리뷰글에 지친 사람
- 지역 맛집을 미리 알아둔 뒤에, 해당 지역을 방문했을때 근처로 오면 알림을 받고 싶은 사람
